Global Economy Explorer - Site Plan

Site Name:

Global Economy Explorer

This name was selected because the website explores economic data and stories from countries around the world, making it clear and descriptive for the user.

Site Purpose

The site provides an interactive platform to explore country-specific economic data, trends, key indicators, and stories behind major events affecting GDP and other economic measures. Users can compare economies, view historical trends, and explore interactive visualizations to understand global economic patterns.

Scenarios

  • Which years had the biggest impact on China's GDP and why?
  • How does Germany’s unemployment trend compare to the US over the last 20 years?

Color Scheme

  • Primary Color: #0d47a1 mainly used for buttons
  • Secondary/ heading Background Color: #121c2a main background for sections headings
  • Teritiary/sections background color: #1E2A3A background for section cards
  • Text Color: #e0e0e0
  • Accent Color: #1976d2 for hover states

Typography

  • Roboto, sans-serif: used for body text and general content.
  • Montserrat, sans-serif: used for headings and section titles.
